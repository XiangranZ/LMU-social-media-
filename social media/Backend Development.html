Backend Development

Backend Development involves working on the server-side of the application, which includes tasks such as maintaining the server-side logic, databases, application infrastructure and creating APIs (Application Programming Interfaces).

APIs (Application Programming Interfaces)

An API is a type of software interface. Different from a user interface that connects a computer to a person, an API allows different parts of the application to communicate with each other. APIs typically follow a request-response model, where one application sends a request to another application using predefined methods and parameters. In addition, APIs often use standard protocols to facilitate communication between applications, for example HTTP/HTTPs.

Case Study: Instagram

The API provided by Instagram is the Instagram Graph API. It allows developers to access and interact with data on the Instagram platform in a graph-like structure.

HTTP Methods

The HTTP method of an application refers to the type of action requested by a client (such as a web browser or a mobile app) when communicating with a server over the Hypertext Transfer Protocol (HTTP).

HTTP Methods of Instagram

POST Upload media, add comments
Input: User-ID, media content
Output: Information about the uploaded media
PUT Edit caption on post, edit comment
Input: Media-ID
Output: Information about the added comment
DELETE delete comment on a post, delete post, delete profile
Input: Media-ID, comment-ID
Output: Confirmation of the deleted comment
GET User profile, retrieve media, search media by hashtag, get comments on a post, get followers/following list, retrieve stories and posts
Input: User-ID
Output: Username, profile picture, follower/following count
