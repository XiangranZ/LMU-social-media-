Data Storage and Databases

Data Storage

Data storage refers to the physical or digital means of preserving and retaining data for future use. It includes various storage technologies, such as hard disk drives (HDDs), solid-state drives (SSDs), cloud storage, tape storage, optical storage, and more..

Databases

A database is a structured and organized collection of data for efficient storage, retrieval, and management. It is designed to support the efficient querying and processing of data, enforcing relationships between different pieces of information.

Types of Database

Relational Databases (RDBMS)

  • Organize data into tables with rows and columns.
  • Enforce relationships between tables.
  • Examples: MySQL, PostgreSQL, Oracle Database, Microsoft SQL Server.

NoSQL Databases

  • Designed for flexibility and scalability.
  • Do not rely on a fixed schema.

In-Memory Databases

  • Store data in the system's main memory (RAM) for faster access.

Distributed Databases

  • Distribute data across multiple nodes or servers for improved scalability and fault tolerance.

Case Study: Graph Database

A graph database is a type of NoSQL database that uses graph structures for data storage, with nodes, edges, and properties to represent and store data. Graph databases typically have a flexible schema, allowing nodes and edges to have different types and properties.
