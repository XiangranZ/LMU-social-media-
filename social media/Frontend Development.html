Frontend Development

Frontend development revolves around designing the user interface and user experience (UI/UX) of a website or application. Its objective is to ensure that the application or website is visually appealing, responsive, and user-friendly. Frontend developers engage with the client side of web development.

Key Technologies (web development)

  • HTML (Hypertext Markup Language): structure of contents
  • CSS (Cascading Style Sheets): styling and layout
  • JavaScript: interactivity
