Version Control System

A Version Control System (VCS) is a software tool that helps manage changes to source code and other files over time. It is commonly used in collaborative software development to track and coordinate changes made by multiple contributors to a project.

GIT / Github

Git is a distributed version control system that allows multiple developers to collaborate on a project.

GitHub is a web-based platform that provides hosting for Git repositories. It enhances the functionality of Git by adding a graphical interface, collaboration tools, and additional features
