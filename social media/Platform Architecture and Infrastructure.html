Platform Architecture and Infrastructure

Definitions

Platform Architecture

Platform architecture describes the overall design and structure of a platform, including the arrangement and interaction of components to realize specific functions.

Platform Infrastructure

The underlying framework or physical components, such as hardware, software, and networks, that enable the operation of a platform.

Client/Server Architecture

Client refers to the end-user device or application that interacts with the server. Server refers to the centralized system or software that responds to client requests. Client/server architecture describes the relationship between clients and servers.

Client/Server Infrastructure

Client/Server infrastructure enables communication between clients and servers. It supports the functioning of the architecture.

Teamwork - Buzzwords

Hypertext Transfer Protocol (HTTP)

Hypertext Transfer Protocol (HTTP) is a protocol for data exchange on the internet. It is used to transfer data between a web browser and a web server. When a user enters a URL into a web broswer, the browser sends an HTTP request to the server assoiated with that URL. Subsequently, the server responds with the requested web page or other resources.

TLS (Transport Layer Security)

TLS is a cryptographic protocol that protects internet communications. It creates secure conneitons between a client(e.g. a web browser) and a server.

DNS (Domain Name System)

DNS translates domain names (e.g. wwww.wikipedia.com) into IP addresses that computer uses to identify each other on the internet. It enables web browsers to access websites and other internet resources.
